# Duke Hoang — QA & AI Notes

A personal knowledge base covering QA engineering and AI testing — written by a practitioner, for practitioners.

Live site: [murosaki12.github.io/practicioner_blog](https://Murosaki12.github.io/practicioner_blog)

## Stack

Built with [Astro](https://astro.build) + MDX. Content authored locally via [Keystatic](https://keystatic.com) (disabled during CI builds). Deployed to GitHub Pages via GitHub Actions.

## What's in here

- **Notes** — QA Framework, AI for QA, and more as they come
- **Projects** — things I've built or worked on
- **Changelog** — version history as the site grows

## Local dev

```bash
npm install
npm run dev
```

## Deployment

Pushes to `main` auto-deploy via `.github/workflows/deploy.yml`.

## About

I'm a QA engineer based in Japan working on hardware/software products. This site is where I document what I learn — practical stuff, no fluff.
